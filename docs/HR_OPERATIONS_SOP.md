# HR Operations SOP

Purpose: Daily/weekly/month-end operating guide for HR module users.

---

## 1) Daily Operations

- Verify attendance entry/import completion
- Review pending attendance regularizations
- Review pending leave approvals
- Check HR ESS tickets and escalations
- Track failed actions in logs (if any)

---

## 2) Weekly Operations

- Validate active employee list changes (join/exit/status)
- Review shift/roster correctness
- Validate leave balance anomalies
- Check recruitment pipeline movement
- Review performance cycle pending actions

---

## 3) Month-End Operations

- Freeze attendance for payroll period
- Review and finalize leave balances
- Run payroll calculation and review exceptions
- Approve and post payroll
- Validate posted payroll voucher totals with Finance
- Publish payslips

---

## 4) Incident Handling

- Classify defect severity: Critical/High/Medium/Low
- For Critical payroll or security issues:
  - stop new postings
  - inform technical owner immediately
  - capture exact time, user, and screenshot
- Re-test after fix and record closure evidence

---

## 5) Access and Security

- Use least-privilege role assignments
- Do not share accounts
- Quarterly role/access review mandatory
- Report any tenant data visibility anomaly immediately
