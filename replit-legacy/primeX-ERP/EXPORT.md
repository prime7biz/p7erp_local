# Export / Archive Instructions

When exporting or zipping this project, **exclude** the following:

| Path | Reason |
|------|--------|
| `.env` | Contains secrets (DB credentials, API keys) |
| `.env.*` | Environment-specific secret overrides |
| `.git/` | Version-control history — large, not needed in archives |
| `node_modules/` | Regenerated by `npm install` |
| `dist/` | Regenerated by `npm run build` |

## Recommended export command

```bash
zip -r prime7-export.zip . -x ".env" ".env.*" ".git/*" "node_modules/*" "dist/*"
```

Or use Replit's **"Download as zip (source)"** option, which excludes `.git` and `node_modules` automatically. Verify `.env` is not included before sharing.
