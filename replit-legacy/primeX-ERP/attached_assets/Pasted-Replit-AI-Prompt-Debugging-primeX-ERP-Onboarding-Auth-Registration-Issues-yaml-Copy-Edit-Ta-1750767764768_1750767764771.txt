## 🔐 Step 1: Fix Cookie-based Auth (Cross-Origin)

### In `POST /api/auth/login` route:
Ensure cookie is set like this:
```ts
res.cookie('token', jwtToken, {
  httpOnly: true,
  secure: true,
  sameSite: 'None', // Needed for cross-site requests
  maxAge: 24 * 60 * 60 * 1000, // 1 day
});
